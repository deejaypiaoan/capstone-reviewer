Project Overview

System Description

The ASAP CKD Predictor is a comprehensive healthcare analytics platform designed for chronic kidney disease (CKD) patient data management, risk assessment, and survival analysis. It combines Laravel PHP framework with Python FastAPI machine learning services to provide clinical decision support tools for healthcare professionals.

Key Components:
  • Patient Management System with CRUD operations
  • Machine Learning-based Risk Assessment (97% accuracy)
  • Survival Analysis using Cox Regression Models
  • Interactive Analytics Dashboard
  • Data Export Features (CSV/PDF)
  • Role-based Authentication System

Purpose & Objectives

Purpose of this Study

The main goal of this study was to create and test the ASAP CKD Predictor system that helps healthcare workers analyze patient data and check risks for people with kidney disease. The researchers wanted to see how well the system works for managing patient information and making predictions about chronic kidney disease.

Objectives of the Study
  1. Determine the current patient data management processes and CKD prediction methodologies used at ASAP Tagudin Medical Diagnostic Center.
  2. Identify the features integrated into the developed system for CKD prediction, survival analysis, and patient data visualization.
  3. Assess the level of usability of the proposed system using the System Usability Scale (SUS) questionnaire.

Research Gaps

Identified Gap

The lack of easy-to-use tools for analyzing outcomes, finding survival patterns, and assessing risks limits how well healthcare workers and researchers can understand CKD in their areas. This gap affects both clinical practice and research.

Specific Problems:
  • Manual patient data management using paper records and simple spreadsheets
  • No systematic approach for CKD risk assessment
  • Limited access to modern analysis tools
  • Difficulty in identifying patterns and trends in patient health
  • Inability to generate comprehensive reports and insights
How Our Project Fills the Gap:

Our ASAP CKD Predictor system provides a comprehensive web-based platform that integrates machine learning algorithms with interactive data visualization to enable healthcare professionals to make evidence-based clinical decisions and improve patient care outcomes.

Possible Important Questions

What is the accuracy of the machine learning models used in this system?

The system achieves 97% accuracy in CKD prediction using Gradient Boosting algorithms. The machine learning models include:

  • Risk Assessment Model: Random Forest Classifier for predicting patient risk levels
  • Survival Analysis Model: Cox Proportional Hazards Model for survival probability predictions
  • Feature Engineering: 12 clinical parameters including age, gender, creatinine, BUN, hemoglobin, albumin, phosphorus, calcium, diabetes, hypertension, CVD, and dialysis years
How does the authentication system work and what are the different user roles?

The system implements a dual-role authentication system:

  • Super Admin: Full system access including patient CRUD operations, export features, user management, and system configuration
  • Admin: Limited access with view-only patient records, CKD prediction features, analytics dashboard, and basic reporting

Authentication uses Laravel's built-in system with role-based middleware for access control and session management.

What research methodology was used in developing this system?

The project used a Descriptive Developmental Research Design with the Rapid Application Development (RAD) Model:

  • Requirements Planning: Gathered permission from ASAP Tagudin Medical Diagnostic Center
  • User Design: Determined software functionality and system requirements
  • Construction: Built using Laravel PHP, MySQL, Python FastAPI, Bootstrap CSS, Chart.js, and JavaScript
  • Cutover: Conducted practice runs and system testing
How was the system's usability evaluated and what were the results?

The system's usability was evaluated using the System Usability Scale (SUS) questionnaire administered to 6 healthcare professionals at ASAP Tagudin Medical Diagnostic Center (1 Head Nurse, 5 Nursing Staff).

Results:

  • SUS Score: 73.33
  • Grade: B (Good)
  • Standard Deviation: 15.42
  • Score Range: 35-90

This indicates that healthcare workers found the system easy to use and helpful for patient data management and clinical decision-making.

What are the main features of the ASAP CKD Predictor system?

The system includes five main feature modules:

  • Patient Management Module: Complete CRUD operations for patient records, including demographics, medical history, and lab results
  • CKD Prediction Interface: Machine learning-based risk assessment and survival analysis with real-time predictions
  • Analytics Dashboard: Interactive visualizations including patient demographics, survival curves, and risk factor analysis
  • Reporting Features: Comprehensive reports and data export capabilities in CSV and PDF formats
  • Educational Resources: Information about CKD management, treatment methods, and best practices for healthcare workers
What challenges were encountered during development and how were they addressed?

Key challenges included:

  • Data Integration: Successfully integrated real patient data while maintaining privacy and security standards
  • Technical Complexity: Implemented seamless communication between Laravel PHP backend and Python FastAPI ML service
  • User Experience: Designed intuitive interface for healthcare professionals with varying technical expertise
  • Model Accuracy: Achieved 97% prediction accuracy while ensuring clinical relevance and interpretability

These challenges were addressed through iterative development, continuous user feedback, and rigorous testing protocols.

What is the significance of this project for healthcare delivery in the Philippines?

This project addresses critical healthcare challenges in the Philippines:

  • Scale of CKD Problem: CKD affects 11.2% of Filipinos (12.94 million people), with one Filipino developing kidney failure every hour
  • Economic Impact: Total cost of CKD in the Philippines is P592.15 billion, with diabetes patients making up 60.9% of these costs
  • Healthcare Gap: Rural areas like Tagudin, Ilocos Sur lack modern analysis tools for CKD patient management
  • Solution Impact: Provides evidence-based decision support, improves clinical outcomes, and enhances healthcare delivery efficiency

The system fills the gap in analyzing kidney disease outcomes and supports better healthcare delivery for hemodialysis patients.
